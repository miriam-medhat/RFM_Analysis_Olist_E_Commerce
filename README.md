## 📊 Project Overview

This project focuses on performing **RFM (Recency, Frequency, Monetary) Analysis** on the Olist e-commerce dataset to gain deeper insights into customer purchasing behavior. The dataset, stored in a SQLite database, contains transactional data from March 2016 to August 2018 and represents real-world interactions between customers and an online marketplace.

The main objective of this analysis is to segment customers into meaningful groups based on their activity and spending patterns. By doing so, the project helps identify high-value customers, detect those at risk of churning, and support the development of targeted marketing and retention strategies. The project combines SQL for data extraction and Python for analysis, modeling, and visualization.

---

## 🧩 Project Parts

### 🔹 Part 1: SQL Analysis
In this part, SQL is used to explore and extract key insights from the dataset. Queries are written to analyze order trends, revenue distribution, customer spending, product popularity, and review scores. This step builds a strong foundation for understanding the data before moving into deeper analysis.

---

### 🔹 Part 2: Python Data Analysis
This part involves loading the data into Python using Pandas for preprocessing and exploratory data analysis. RFM metrics (Recency, Frequency, Monetary) are calculated for each customer, and customers are segmented based on their scores. Visualizations, such as treemaps, are created using Plotly to clearly represent customer segments and highlight high-value groups.

---

### 🔹 Part 3: Advanced Analysis & Business Strategy
In the final part, insights from the RFM analysis are used to develop business strategies. This includes designing customer retention plans for "At Risk" and "Lost" segments, interpreting visualizations like bubble charts, discussing churn prediction, and suggesting improvements to the RFM segmentation approach for better accuracy in real-world scenarios.
